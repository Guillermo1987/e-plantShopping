# e-plantShopping

## Descripción del Proyecto

e-plantShopping es una aplicación web de comercio electrónico para Paradise Nursery, una tienda de plantas en línea. La aplicación permite a los usuarios navegar por diferentes categorías de plantas, ver detalles de productos, agregar plantas al carrito de compras y gestionar sus compras.

## Características Principales

- Página de inicio con información sobre Paradise Nursery
- Catálogo de plantas organizadas por categorías
- Sistema de carrito de compras con Redux
- Funcionalidad para agregar, incrementar, decrementar y eliminar productos
- Cálculo automático del total de la compra
- Interfaz de usuario intuitiva y responsive

## Tecnologías Utilizadas

- React
- Redux Toolkit
- CSS
- JavaScript (ES6+)

## Objetivos del Proyecto

Crear una experiencia de compra en línea fluida para los amantes de las plantas, permitiéndoles explorar y adquirir plantas para embellecer sus hogares y jardines.
